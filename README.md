# FTP ZIP Unzipper

A multilingual PHP script that allows you to extract ZIP files on your web server via a web interface. Perfect for uploading and extracting website files when your FTP client doesn't support ZIP extraction.

![PHP](https://img.shields.io/badge/PHP-7.4+-777BB4.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Languages](https://img.shields.io/badge/Languages-5-blue.svg)

## Features

- 🌍 **Multilingual Support** - English, Dutch, German, French, Spanish
- 📦 **Multiple ZIP Selection** - Choose from all ZIP files in directory
- 🎨 **Modern UI** - Responsive design with progress indicators
- 🔒 **Security Features** - File validation and cleanup options
- 📱 **Mobile Friendly** - Works on all devices
- 🗑️ **Auto Cleanup** - Remove ZIP files and script after extraction

## Requirements

- PHP 7.4 or higher
- ZipArchive extension enabled
- Web server with file write permissions
- FTP access to upload files
